# Kiwi Portfolio - Static Version

A fast, modern, and visually engaging static version of my personal portfolio, built with standard web technologies and polished with custom interactions.

## 🛠 Technologies Used

- **HTML5**: Semantic and accessible structure.
- **TailwindCSS (CDN)**: For rapid, responsive, and modern utility-first styling.
- **CSS3 (Custom)**: Custom animations, glassmorphism effects, and stability fixes for the UI.
- **JavaScript (Vanilla)**: High-performance logic for animations and dynamic content without external heavy libraries.

## 📂 Project Structure

- `index.html`: The main landing page featuring "About Me" and the interactive timeline.
- `contact.html`: Contact page with a dynamic location-based time widget.
- `css/style.css`: Custom styles for 3D effects, animations, and UI refinements.
- `js/script.js`: Core logic for scroll reveals, tilt effects, and real-time updates.
- `public/icons/`: Static assets, including icons and branding images.

## 🚀 How to View the Project

Simply open the `index.html` file in any modern web browser. Since this is a static project, no local server or complex configuration is required.

## ✨ Key Features

- **Interactive Timeline**: A "Journey" section that reveals milestones gracefully as you scroll using `IntersectionObserver`.
- **3D Skill Cards**: Interactive skills grid with a dynamic tilt effect that responds to mouse movement.
- **Dynamic Time Widget**: A custom-built widget on the Contact page that displays the current time in Rome (Europe/Rome) and changes the UI (Dawn, Day, Sunset, Night) and greetings based on the hour.
- **Smart Auto-Scroll**: An intuitive smooth-scroll trigger on the first user interaction to guide them through the experience.
- **Responsive & Modern Design**: Fully optimized for all screen sizes, featuring a clean, professional aesthetic with blue and slate accents.
- **Clean Code Architecture**: Strict separation of concerns between structure, style, and behavior for better maintainability.
