# Kiwi Portfolio - Static Version

Una versione statica, veloce e moderna del mio portfolio personale, costruita con tecnologie web standard.

## 🛠 Tecnologie Utilizzate

- **HTML5**: Struttura semantica.
- **TailwindCSS (CDN)**: Per uno styling rapido, responsivo e moderno.
- **CSS3 (Custom)**: Animazioni personalizzate e stabilità per la griglia delle skills.
- **JavaScript (Vanilla)**: Per effetti di scroll reveal e interazioni dinamiche.

## 📂 Struttura del Progetto

- `index.html`: La home page principale.
- `about.html`: Pagina dettagliata con timeline e skills.
- `css/style.css`: Stili personalizzati per animazioni e fix UI.
- `js/script.js`: Logica per le interazioni e gli effetti di visualizzazione.
- `public/icons/`: Risorse statiche, icone e immagini.

## 🚀 Come Visualizzare il Progetto

Basta aprire il file `index.html` in un qualsiasi browser moderno. Non è necessaria alcuna configurazione o server locale, trattandosi di un sito statico.

## ✨ Caratteristiche

- **Timeline Animata**: Effetto di comparsa graduale al passaggio dello scroll.
- **Skills Interattive**: Griglia di competenze con schede animate al passaggio del mouse.
- **Design Responsivo**: Ottimizzato per desktop e dispositivi mobile.
- **Clean Code**: Separazione delle responsabilità tra HTML, CSS e JS.
